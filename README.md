# speed-run

A speedrun is a play-through (or a recording thereof, namely run) of a video game performed with the intention of completing it as fast as possible within one's own ability. [Wikipedia](https://en.wikipedia.org/wiki/Speedrun)

## Getting Started

iOS application that shows the world record speedruns from [speedrun.com](www.speedrun.com).

### API

We use the speedrun REST API: [speedrun.com/api/v1](https://www.speedrun.com/api/v1/games)

### Prerequisites

Navigate through the terminal to the initial structure of the project

Install all dependencies

```
pod update
```
